Given the head of a singly linked list, determine whether the list is a palindrome (reads the same forward and backward).
 Approach:
Used the slow and fast pointer technique to find the middle of the linked list.
Reversed the second half of the list.
Compared the first half with the reversed second half.
If all elements matched, the list is a palindrome.
Key Concepts:
Linked List traversal
Two-pointer technique
In-place reversal
Complexity:
Time Complexity: O(n)
Space Complexity: O(1)
Outcome:Successfully implemented an optimal solution without using extra space.
